COMO CARREGAR NO CHROME / EDGE / OPERA:
1. Descompacte este ficheiro ZIP para uma pasta no seu computador.
2. Aceda a: chrome://extensions ou edge://extensions
3. Active o "Modo do programador" no canto superior direito.
4. Clique no botão "Carregar sem compactação" (Load unpacked).
5. Selecione a pasta onde descompactou o ficheiro (a pasta que contém o manifest.json).

---
COMO COMPILAR PARA SAFARI (XCODE):
A pasta "SafariNative_Wrapper" contém o esqueleto nativo plist e handlers em Swift.
Para gerar a aplicação macOS / iOS nativa:
1. Abra o terminal no macOS.
2. Corra o comando de conversão:
   xcrun safari-web-extension-converter /caminho/para/pasta_descompactada
